# User Manual

## Getting Started

### Logging In

1. Open browser to https://test-protocols.company.com
2. Enter username and password
3. Click "Login"

### Dashboard Overview

The main dashboard shows:
- **Active Service Requests**: Current projects in progress
- **Pending Tasks**: Protocols assigned to you
- **Recent Completions**: Recently finished tests
- **Equipment Status**: Available testing equipment
- **Alerts**: Important notifications

## Service Request Management

### Creating a Service Request

1. Click **"New Service Request"** button
2. Fill in customer information:
   - Customer name
   - Contact person
   - Email and phone
3. Enter module details:
   - Manufacturer
   - Model number
   - Power rating
   - Quantity
4. Select testing requirements:
   - Check required standards (IEC 61215, IEC 61730, etc.)
   - Or select individual protocols
5. Click **"Submit"**

### Tracking Service Requests

1. Navigate to **"Service Requests"** menu
2. View list of all service requests
3. Use filters:
   - Status
   - Customer
   - Date range
4. Click on SR number to view details

## Protocol Execution

### Starting a Protocol

1. Go to **"My Tasks"**
2. Select assigned protocol
3. Review protocol details
4. Click **"Start Execution"**
5. Verify equipment calibration
6. Enter test parameters
7. Click **"Begin Test"**

### Data Entry

1. Enter measurements as prompted
2. System validates data in real-time
3. Attach photos if required
4. Add notes or observations
5. Click **"Save Progress"** regularly

### Completing a Protocol

1. Review all entered data
2. Complete final measurements
3. Click **"Complete Test"**
4. System performs QC checks
5. Review pass/fail results
6. Add final comments
7. Click **"Submit for Review"**

## Generating Reports

1. Open completed service request
2. Click **"Generate Report"**
3. Select report type:
   - Executive Summary
   - Technical Report
   - Certificate
4. Choose format (PDF/Excel)
5. Include options:
   - Photos
   - Raw data
   - Charts
6. Click **"Generate"**
7. Download when ready

## Common Tasks

### Visual Inspection

1. Receive samples
2. Open **"Visual Inspection"** protocol
3. Follow checklist:
   - Check for cracks
   - Look for delamination
   - Inspect junction box
   - Verify labels
4. Take photos of all sides
5. Record any defects
6. Submit inspection report

### Running STC Power Measurement

1. Ensure simulator is warmed up
2. Open PVTP-001 protocol
3. Enter module serial number
4. Place module on test bed
5. Connect leads
6. Click **"Measure"**
7. System captures I-V curve
8. Review results
9. Accept or retry measurement

## Troubleshooting

### Equipment Not Available

- Check equipment calendar
- Contact lab manager
- Reschedule test

### Measurement Errors

- Verify equipment calibration
- Check connections
- Review environmental conditions
- Retry measurement

### Data Entry Issues

- Contact IT support
- Use backup form
- Document issue
