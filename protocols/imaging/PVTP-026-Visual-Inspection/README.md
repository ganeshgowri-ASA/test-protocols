# PVTP-026: Visual Inspection Protocol

## Overview
Automated visual inspection using high-resolution imaging and computer vision AI/ML for detecting cracks, discoloration, bubbles, and other physical defects.

## Defect Types
- Cell & glass cracks
- Discoloration & yellowing
- Bubbles & delamination
- Scratches & chips
- Frame damage
- Junction box issues

## AI/ML Models
- YOLOv8 object detection
- U-Net++ segmentation
- Color space analysis

## Standards
- IEC 61215, IEC 61730
- ASTM E2481
- ISO 9001:2015
