# WET-001: Wet Leakage Current Test Protocol

## Overview

**Protocol ID**: WET-001
**Standard**: IEC 61730 MST 02
**Category**: Safety
**Version**: 1.0.0
**Status**: Active

## Purpose

The Wet Leakage Current Test evaluates the electrical safety of photovoltaic (PV) modules under humid conditions. This test ensures that modules maintain adequate insulation and do not pose electrical safety hazards when exposed to moisture.

## Standard Reference

IEC 61730-2:2016 Module Safety Testing (MST) 02 - Wet Leakage Current Test

This test is part of the IEC 61730 standard for PV module safety qualification, which ensures modules meet safety requirements for protection against electric shock, fire, and mechanical hazards.

## Test Principle

The test applies a DC voltage to the PV module while subjecting it to high humidity conditions. Leakage current is continuously monitored to detect any degradation in insulation that could lead to electric shock hazards or fire risks.

## Test Conditions

### Environmental Requirements

| Parameter | Specification | Tolerance |
|-----------|--------------|-----------|
| Temperature | 20-30°C | ±5°C |
| Relative Humidity | 85-95% RH | ±5% |
| Barometric Pressure | 80-110 kPa | N/A |

### Test Parameters

| Parameter | Default Value | Range |
|-----------|--------------|-------|
| Test Voltage | 1500 V DC | 0-5000 V |
| Test Duration | 168 hours (1 week) | 1-720 hours |
| Measurement Interval | 60 minutes | 1-1440 minutes |
| Electrode Configuration | Type A or B | A, B |
| Voltage Polarity | Positive or Negative | +/- |

### Electrode Configurations

**Type A (Direct Configuration)**:
- Test voltage applied directly to module terminals
- Simulates normal operating conditions

**Type B (Simulated Configuration)**:
- Test voltage applied through water film
- Simulates extreme moisture exposure

## Acceptance Criteria

### Primary Criteria

1. **Maximum Leakage Current**: ≤ 0.25 mA
2. **Minimum Insulation Resistance**: ≥ 400 MΩ

### Secondary Criteria

3. **Voltage Stability**: Variation ≤ 5%
4. **No Surface Tracking**: Visual inspection shows no tracking
5. **No Visible Damage**: No cracks, delamination, or burn marks

## Test Procedure

### 1. Pre-Test Preparation

1. Visual inspection of module
2. Record module identification data
3. Verify test equipment calibration
4. Set environmental chamber conditions
5. Allow module to stabilize (minimum 2 hours)

### 2. Test Setup

1. Position module in environmental chamber
2. Configure electrodes per specification
3. Connect measurement equipment
4. Verify environmental conditions
5. Initialize data logging system

### 3. Test Execution

1. Start environmental conditioning
2. Apply test voltage gradually
3. Begin continuous monitoring:
   - Leakage current (every measurement interval)
   - Voltage
   - Temperature
   - Humidity
4. Record any anomalies or observations
5. Continue for specified test duration

### 4. Test Completion

1. Gradually reduce test voltage to zero
2. Disconnect measurement equipment
3. Remove module from chamber
4. Visual inspection for damage
5. Record final observations

### 5. Data Analysis

1. Calculate statistical measures:
   - Maximum, minimum, average leakage current
   - Standard deviation
   - Insulation resistance values
2. Check against acceptance criteria
3. Detect anomalies
4. Perform trending analysis
5. Determine pass/fail status

### 6. Reporting

1. Generate automated test report
2. Include:
   - Test parameters and conditions
   - Measurement data and charts
   - Statistical analysis
   - Pass/fail determination
   - Observations and notes
3. Archive test data

## Measurements

### Data Collection

The following parameters are measured at each interval:

| Parameter | Unit | Range | Precision |
|-----------|------|-------|-----------|
| Leakage Current | mA | 0-10 | ±0.0001 |
| Applied Voltage | V | 0-5000 | ±1 |
| Temperature | °C | 15-35 | ±0.1 |
| Relative Humidity | % | 45-100 | ±1 |
| Insulation Resistance | MΩ | 0-10000 | ±1 |

### Calculated Values

- **Insulation Resistance**: R = V / I
  - Where V = applied voltage (V)
  - I = leakage current (A)

- **Voltage Deviation**: |V_measured - V_nominal| / V_nominal × 100%

## Analysis Methods

### Statistical Analysis

1. **Descriptive Statistics**:
   - Mean, median, mode
   - Standard deviation
   - Min/max values
   - Range and quartiles

2. **Trend Analysis**:
   - Linear regression
   - Rate of change
   - Direction (increasing/decreasing/stable)

3. **Anomaly Detection**:
   - Z-score method (threshold: 3σ)
   - Identification of outliers
   - Root cause analysis

### Pass/Fail Determination

Test **PASSES** if:
- ✓ Max leakage current ≤ 0.25 mA
- ✓ Min insulation resistance ≥ 400 MΩ
- ✓ Voltage variation ≤ 5%
- ✓ No surface tracking observed
- ✓ No visible damage

Test **FAILS** if any criterion is not met.

## Equipment Requirements

### Test Equipment

1. **DC Power Supply**:
   - Voltage range: 0-5000 V
   - Current capability: ≥ 10 mA
   - Stability: ±1%

2. **Current Measurement**:
   - Range: 0-10 mA
   - Resolution: 0.0001 mA
   - Accuracy: ±0.1%

3. **Environmental Chamber**:
   - Temperature: 15-35°C (±0.5°C)
   - Humidity: 45-100% RH (±2%)
   - Capacity: Module size + 20%

4. **Data Acquisition System**:
   - Sampling rate: ≥ 1 Hz
   - Channels: ≥ 6
   - Automated logging

### Calibration Requirements

All equipment must be calibrated within 12 months:
- DC power supply: ±0.5%
- Current meters: ±0.1%
- Temperature sensors: ±0.1°C
- Humidity sensors: ±1% RH

## Troubleshooting

### Common Issues

| Issue | Possible Cause | Solution |
|-------|---------------|----------|
| High leakage current | Poor insulation | Check module integrity |
| Unstable readings | Environmental fluctuation | Verify chamber stability |
| Equipment alarm | Overcurrent/overvoltage | Check connections, reduce voltage |
| No data logging | Software/hardware issue | Restart DAQ system |
