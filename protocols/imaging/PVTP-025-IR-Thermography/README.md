# PVTP-025: Infrared (IR) Thermography

## Overview
Advanced IR thermography protocol for detecting thermal anomalies, hotspots, and electrical defects in solar modules with AI/ML-powered thermal pattern recognition.

## Features
- ✅ Radiometric thermal imaging with atmospheric corrections
- ✅ AI/ML hotspot detection and classification
- ✅ Automated severity assessment
- ✅ Real-time environmental monitoring
- ✅ EL-IR data fusion capabilities
- ✅ Full traceability and safety monitoring

## Detectable Thermal Anomalies
- Bypass diode activation
- Cell hotspots
- Junction box overheating
- String mismatch
- Shading effects
- Cell failures
- Interconnect resistance issues

## Standards Compliance
- IEC 62446-3
- ASTM E1934, E1933
- ISO 9712
